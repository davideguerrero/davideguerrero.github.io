MTO by
FENDI

MTO by FENDI is an application designed to help FENDI's sales assistants in Made To Order events, where they have to create unique designs directly with the client. The objective of this project was to redesign and prototype the UI / UX of the application to make it easy to use for the sales assistants and the clients too.
